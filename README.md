# Teslo API

1. Clonar proyecto
2. ```pnpm i```
3. Clonar el archivo ```.env.temlat``` y renombrarlo a ```.env```
4. Cambiar  las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```

6. Ejecutar Seed
```
http://localhost:300/api/seed
```

7. Levantar modo de desarrollo
```
pnpm start:dev
```
